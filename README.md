# JAVASCRIPT-Concatenate


Concatenate a string couple of ways:<br>
Join with variables

    const word1 = 'hello';
    const word2 = 'world!';
    
    console.log(word1 + ' ' + word2); // output: 'hello world!'

Interpolate using template literal (Important backticks)

    console.log(`${word1} ${word2}`); // output: 'hello world!'
