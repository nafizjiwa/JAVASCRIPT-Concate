# JAVASCRIPT-Concate


Couple of ways to concatenate a string:

    const word1 = 'hello';
    const word2 = 'world!';
    
    console.log(word1 + ' ' + word2); // output: 'hello world!'

We can also interpolate it using a template literal, but remember to use backticks!

    console.log(`${word1} ${word2}`); // output: 'hello world!'
